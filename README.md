# react-redux-axios-boilerplate

### Install dependencies.

```
> npm install
```

### Run react app

```
> npm start
```

#### Run mock api

```
> npm run api
```

## Production build can be tested with [local-web-server](https://github.com/75lb/local-web-server)


```
> npm run build
> cd ./dist
> ws -s index.html
```
